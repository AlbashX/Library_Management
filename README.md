# 📚 Library Management System

A modern **web-based Library Management System** that allows students to browse, search, and download academic books easily.  
The platform provides an organized digital library where users can filter books by academic year and semester, view details, and access downloadable study materials.

🔗 Live Website  
https://albashx.github.io/Library_Management/

---

# 🚀 Features

• Browse a collection of academic books  
• Search books by title or author  
• Filter books by **Year** and **Semester**  
• View detailed book descriptions  
• Download or read books directly via Google Drive links  
• Responsive modern UI using Bootstrap  
• Interactive modal popups for book details  
• Clean and structured UI design for easy navigation

# ⚙️ How It Works

The system uses a **JavaScript-based book database** stored inside the application.  
Each book contains:

- Title
- Author
- Academic year
- Semester
- Description
- Cover image
- Download link

Books are dynamically rendered into **interactive cards**, allowing users to search, filter, and access resources quickly.
